# C# - Linear Algebra

This project is an introduction to linear algebra and its concepts using C#.

## Topics Covered

- Vectors and vector operations (magnitude, addition, scalar multiplication, dot product, cross product)
- Matrices and matrix operations (addition, scalar multiplication, matrix multiplication, rotation, shear, transpose, determinant, inverse)

## Requirements

- All C# files compiled on Ubuntu 14.04 LTS using dotnet
- Target framework: netcoreapp2.1
- All public classes and members have XML documentation tags
