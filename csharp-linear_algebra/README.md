# C# Linear Algebra

A C# project implementing linear algebra operations including vector and matrix mathematics.

## Tasks

- Vector magnitude (2D and 3D)
- Vector addition (2D and 3D)
- Vector-scalar multiplication (2D and 3D)
- Dot product (2D and 3D)
- Cross product (3D)
- Matrix addition
- Matrix-scalar multiplication
- Matrix-matrix multiplication
- Matrix rotation (2D)
- Matrix shear (2D)
- Matrix transpose
- Determinant calculation (2x2 and 3x3)
- Matrix inverse (2x2)
