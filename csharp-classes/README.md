# C# - Classes and Namespaces

This project covers C# object-oriented programming fundamentals including classes, namespaces, fields, constructors, properties, methods, and the `ToString()` override.

## Tasks

- **0-enemy**: Empty `Zombie` class in `Enemies` namespace
- **1-enemy**: `Zombie` with public `health` field and default constructor
- **2-enemy**: `Zombie` with health validation constructor
- **3-enemy**: `Zombie` with private `health` field and `GetHealth()` method
- **4-enemy**: `Zombie` with private `name` field and `Name` property
- **5-enemy**: `Zombie` with `ToString()` override
