# C# - Structs, Enumerations

This project contains exercises on structs and enumerations in C#.

## Learning Objectives

- What is a struct
- What is a constructor
- What is a field
- What is a property
- What is an enumeration and when to use them
- What does toString do and how to override it

## Requirements

- Allowed editors: Visual Studio Code
- All files will be compiled on Ubuntu 14.04 LTS using dotnet
- Each C# task requires its own folder and .csproj file

## Tasks

### 0. They're good dogs
Define a new enum Rating with the values Good, Great, Excellent.

### 1. Chief Puppy Officer
Define a new struct Dog with public fields: name, age, owner, and rating.

### 2. A dog is the only thing on earth that loves you more than you love yourself
Add a constructor to struct Dog that takes parameters.

### 3. A dog will teach you unconditional love
Override the .ToString() method to print the Dog object's attributes.
